## How to use a skill in ChatGPT

Paste the skill into the conversation, or host the raw markdown somewhere public and say:

Load this skill and use it for the rest of this conversation.

Then add your task, for example:

Design a landing page hero for a SaaS product. Keep it premium, modern, and WCAG-aware.
